# 使用文档

## 1. 项目简介

本项目基于海康 `HCNetSDK` 的 Windows 64 位库，使用 Python `ctypes` 封装以下能力：

- SDK 初始化与清理
- 设备登录与登出
- 双向语音对讲
- 语音转发
- 实时取流录像
- 语音输入/输出音量调节
- 复合流录像前的 ISAPI 能力检查与配置切换
- 录制视频音频分析

## 2. 运行环境

- Windows 64 位
- Python 3
- 项目自带 `libs/win64` 目录中的海康 SDK 动态库
- 视频分析依赖本机安装 `ffmpeg`

## 3. 目录说明

- `hikvision_voice.py`：核心 SDK 封装
- `hikvision_isapi.py`：ISAPI 配置、音量控制和复合流能力封装
- `video_analysis.py`：录制视频分析能力
- `demo_voice_call.py`：命令行对讲示例
- `demo_stream_record.py`：命令行取流录像示例
- `demo_composite_stream_record.py`：复合流录像示例
- `demo_video_analysis.py`：录制视频分析示例
- `libs/win64`：海康 SDK 依赖库
- `SdkLog/`：SDK 运行日志目录

## 4. 快速开始

### 4.1 命令行对讲

```powershell
python .\demo_voice_call.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345
```

### 4.2 开启 SDK 日志

```powershell
python .\demo_voice_call.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345 `
  --enable-log
```

## 5. 普通录像

### 5.1 命令行录像

```powershell
python .\demo_stream_record.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345 `
  --duration 60
```

默认输出路径：

`recordings/streams/<device-ip>/stream_ch<channel>_YYYYMMDD_HHMMSS.mp4`

### 5.2 代码调用录像

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

recorder = sdk.start_stream_record(session=session)

try:
    input("录像中，按回车结束...")
finally:
    recorder.stop()
    sdk.logout(session)
    sdk.cleanup()
```

## 6. 复合流录像

### 6.1 命令行复合流录像

```powershell
python .\demo_composite_stream_record.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345
```

默认行为：

- 先将语音输入、输出音量调到最大
- 查询 `/ISAPI/Streaming/channels/<trackStreamID>/capabilities`
- 判断设备是否支持复合流录像
- 获取当前流配置
- 若 `./Audio/enable` 或 `./Audio/enabled` 不是 `true`，则自动修改并 PUT 回设备
- 开始录像
- 默认录像时长为 `10` 秒
- 录像文件保存到 `recordings/streams/<device-ip>/`

### 6.2 代码调用复合流切换与音量控制

```python
from hikvision_isapi import HikvisionIsapiClient
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
isapi = HikvisionIsapiClient(sdk)
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

volume_status = isapi.ensure_audio_input_output_volume_max(session=session)
print(volume_status)

status = isapi.ensure_composite_stream_recording_enabled(session=session, channel=1)
print(status)

sdk.logout(session)
sdk.cleanup()
```

## 7. 视频分析

### 7.1 命令行分析

```powershell
python .\demo_video_analysis.py `
  --video recordings\streams\192.168.1.64\stream_ch1_20260529_120000.mp4 `
  --reference-audio recordings\use_cases\192.168.1.64\random_audio_20260529_115900.wav
```

分析内容：

- 从视频中剥离音频
- 判断音频里是否有有效声音
- 判断录制音频中是否包含参考随机音频

说明：

- 默认通过 `ffmpeg` 提取音频
- 如果 `ffmpeg` 不在 PATH，需要通过 `--ffmpeg-path` 传入可执行文件路径

## 8. 语音转发示例

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

def on_audio(data: bytes, audio_flag: int) -> None:
    print("recv bytes:", len(data), "flag:", audio_flag)

forward = sdk.start_voice_forward(session, encoded_audio_callback=on_audio)

try:
    input("语音转发中，按回车结束...")
finally:
    forward.stop()
    sdk.logout(session)
    sdk.cleanup()
```

## 9. 常见说明

- `start_stream_record()` 在未传入 `file_path` 时，会自动保存到 `recordings/streams/<device-ip>/`。
- `set_audio_input_volume_to_max()`、`set_audio_output_volume_to_max()` 和 `ensure_audio_input_output_volume_max()` 位于 `hikvision_isapi.py`，用于将语音输入和输出声量调整到最大。
- `ensure_composite_stream_recording_enabled()` 位于 `hikvision_isapi.py`，用于完成复合流能力检查和音频节点切换。
- 视频分析能力位于 `video_analysis.py`，不耦合设备控制流程，可独立复用。
