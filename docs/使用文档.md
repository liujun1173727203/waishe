# 使用文档

## 1. 项目简介

本项目基于海康 `HCNetSDK` 的 Windows 64 位库，使用 Python `ctypes` 封装以下能力：

- SDK 初始化与清理
- 设备登录与登出
- 双向语音对讲
- 语音转发
- 实时取流录像
- 复合流录像前的 ISAPI 能力检查与配置切换

## 2. 运行环境

- Windows 64 位
- Python 3
- 项目自带 `libs/win64` 目录中的海康 SDK 动态库

## 3. 目录说明

- `hikvision_voice.py`：核心 SDK 封装
- `demo_voice_call.py`：命令行对讲示例
- `demo_stream_record.py`：命令行取流录像示例
- `demo_composite_stream_record.py`：复合流录像示例
- `libs/win64`：海康 SDK 依赖库
- `SdkLog/`：SDK 运行日志目录

## 4. 快速开始

### 4.1 命令行对讲

```powershell
python .\demo_voice_call.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345
```

### 4.2 开启 SDK 日志

```powershell
python .\demo_voice_call.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345 `
  --enable-log
```

日志文件默认输出到当前工作目录下的 `SdkLog/`。

## 5. 代码调用示例

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize(enable_log=True)
sdk.set_talk_mode(use_windows_api=False)

session = sdk.login("192.168.1.64", 8000, "admin", "12345")
call = sdk.start_call(session)

input("语音对讲中，按回车结束...")

call.stop()
sdk.logout(session)
sdk.cleanup()
```

## 6. 取流录像

### 6.1 普通取流录像

```powershell
python .\demo_stream_record.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345 `
  --duration 60
```

默认输出路径：

`recordings/streams/<device-ip>/stream_ch<channel>_YYYYMMDD_HHMMSS.mp4`

### 6.2 代码调用录像

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

recorder = sdk.start_stream_record(session=session)

try:
    input("录像中，按回车结束...")
finally:
    recorder.stop()
    sdk.logout(session)
    sdk.cleanup()
```

## 7. 复合流录像

### 7.1 命令行复合流录像

```powershell
python .\demo_composite_stream_record.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345
```

默认行为：

- 先查询 `/ISAPI/Streaming/channels/<trackStreamID>/capabilities`
- 判断设备是否支持复合流录像相关音频配置
- 获取当前流配置
- 若 `./Audio/enable` 或 `./Audio/enabled` 不是 `true`，则自动修改并 PUT 回设备
- 切换完成后开始录像
- 默认录像时长为 `10` 秒
- 录像文件保存到 `recordings/streams/<device-ip>/`

### 7.2 代码调用复合流切换

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

status = sdk.ensure_composite_stream_recording_enabled(session=session, channel=1)
print(status)

sdk.logout(session)
sdk.cleanup()
```

## 8. 语音转发示例

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

def on_audio(data: bytes, audio_flag: int) -> None:
    print("recv bytes:", len(data), "flag:", audio_flag)

forward = sdk.start_voice_forward(session, encoded_audio_callback=on_audio)

try:
    input("语音转发中，按回车结束...")
finally:
    forward.stop()
    sdk.logout(session)
    sdk.cleanup()
```

## 9. 常见说明

- `start_stream_record()` 在未传入 `file_path` 时，会自动保存到 `recordings/streams/<device-ip>/`。
- `ensure_composite_stream_recording_enabled()` 会先检查设备是否支持复合流录像，再按 ISAPI 将音频节点设置为 `true`。
- 复合流能力判断基于 `/ISAPI/Streaming/channels/<trackStreamID>/capabilities` 返回内容。
- 部分设备返回的音频节点名称可能为 `enable` 或 `enabled`，当前实现两者都兼容。
