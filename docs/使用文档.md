# 使用文档

## 1. 项目简介

本项目基于海康 `HCNetSDK` 的 Windows 64 位库，使用 Python `ctypes` 封装设备语音对讲能力，支持：

- SDK 初始化与清理
- 设备登录与登出
- 对讲模式切换
- 发起双向语音对讲
- 获取和设置设备语音编码参数
- 启动语音转发并收发编码后音频

## 2. 运行环境

- Windows 64 位
- Python 3
- 项目自带 `libs/win64` 目录中的海康 SDK 动态库

## 3. 目录说明

- `hikvision_voice.py`：核心 SDK 封装
- `demo_voice_call.py`：命令行对讲示例
- `demo_stream_record.py`：命令行取流录像示例
- `libs/win64`：海康 SDK 依赖库
- `SdkLog/`：SDK 运行日志目录

## 4. 快速开始

### 4.1 命令行对讲

```powershell
python .\demo_voice_call.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345
```

参数说明：

- `--host`：设备 IP 或主机名
- `--port`：设备 SDK 端口，默认 `8000`
- `--username`：登录用户名
- `--password`：登录密码
- `--voice-channel`：对讲通道，默认 `0`，表示使用设备默认对讲通道
- `--windows-api`：启用旧版 Windows API 对讲模式
- `--pcm-callback`：请求原始 PCM 回调
- `--enable-log`：开启 SDK 日志输出

### 4.2 启用 SDK 日志

```powershell
python .\demo_voice_call.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345 `
  --enable-log
```

日志文件会输出到当前工作目录下的 `SdkLog/`。

## 5. 代码调用示例

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize(enable_log=True)
sdk.set_talk_mode(use_windows_api=False)

session = sdk.login("192.168.1.64", 8000, "admin", "12345")
call = sdk.start_call(session)

input("语音对讲中，按回车结束...")

call.stop()
sdk.logout(session)
sdk.cleanup()
```

## 6. 取流录像示例

### 6.1 命令行录像

```powershell
python .\demo_stream_record.py `
  --host 192.168.1.64 `
  --port 8000 `
  --username admin `
  --password 12345 `
  --duration 60
```

输出文件默认保存到：

`recordings/streams/<device-ip>/stream_ch<channel>_YYYYMMDD_HHMMSS.mp4`

可选参数：

- `--channel`：预览通道，默认 `0`，表示自动使用设备默认预览通道
- `--duration`：录像时长，单位秒
- `--output`：自定义输出文件路径

### 6.2 代码调用录像

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

recorder = sdk.start_stream_record(session=session)

try:
    input("录像中，按回车结束...")
finally:
    recorder.stop()
    sdk.logout(session)
    sdk.cleanup()
```

## 7. 语音转发示例

```python
from hikvision_voice import HikvisionVoiceSDK

sdk = HikvisionVoiceSDK()
sdk.initialize()
session = sdk.login("192.168.1.64", 8000, "admin", "12345")

def on_audio(data: bytes, audio_flag: int) -> None:
    print("recv bytes:", len(data), "flag:", audio_flag)

forward = sdk.start_voice_forward(session, encoded_audio_callback=on_audio)

try:
    input("语音转发中，按回车结束...")
finally:
    forward.stop()
    sdk.logout(session)
    sdk.cleanup()
```

## 8. 常见说明

- `start_call()` 复用 SDK 自带的本地采集和播放能力，不依赖第三方音频库。
- `start_stream_record()` 基于 `NET_DVR_RealPlay_V40` 和 `NET_DVR_SaveRealData`，保存的是设备实时码流录像文件。
- `start_stream_record()` 在未传入 `file_path` 时，会自动保存到 `recordings/streams/<device-ip>/` 目录。
- `start_voice_forward()` 面向编码后音频收发场景，适合接入自定义编解码或转存逻辑。
- 如果设备音频编码参数被修改，通常需要设备重启后生效。
- 如果设备通过 NVR 挂接，默认对讲通道可能不是 `1`，应优先使用登录返回信息中的默认通道。
